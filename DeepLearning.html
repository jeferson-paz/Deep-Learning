def visualiza_imagens(images, labels):
    plt.figure(figsize = (10,10))
    for i in range(25):
        plt.subplot(5, 5, i+1)
        plt.xticks([])
        plt.yticks([])
        plt.grid(False)
        plt.imshow(images[i], cmap = plt.cm.binary)
        plt.xlabel(nomes_classes[labels[i][0]])

In[9]:
# Modelo

# Cria o objeto de sequncia de camadas
modelo_dsa = models.Sequential()

# Adiciona o primeiro bloco de convoluo e max pooling (camada de entrada)
modelo_dsa.add(layers.Conv2D(32, (3, 3), activation = 'relu', input_shape = (32, 32, 3)))
modelo_dsa.add(layers.MaxPooling2D((2, 2)))

# Adiciona o segundo bloco de convoluo e max pooling (camada intermediria)
modelo_dsa.add(layers.Conv2D(64, (3, 3), activation = 'relu'))
modelo_dsa.add(layers.MaxPooling2D((2, 2)))

# Adiciona o terceiro bloco de convoluo e max pooling (camada intermediria)
modelo_dsa.add(layers.Conv2D(64, (3, 3), activation = 'relu'))
modelo_dsa.add(layers.MaxPooling2D((2, 2)))
In[10]:
# Adicionar camadas de classificao
modelo_dsa.add(layers.Flatten())
modelo_dsa.add(layers.Dense(64, activation = 'relu'))
modelo_dsa.add(layers.Dense(10, activation = 'softmax'))

Concluso

Ao longo deste projeto, desenvolvi um modelo de classificao de imagens utilizando Deep Learning com TensorFlow. O modelo foi treinado e testado no dataset CIFAR-10, alcanando uma acurcia de 71,55% nos dados de teste. Utilizando uma rede neural convolucional, consegui identificar e classificar imagens em 10 categorias distintas, como 'airplane', 'automobile', 'bird', entre outras.

O modelo foi posteriormente avaliado com novas imagens, demonstrando sua capacidade de generalizar e realizar previses em dados inditos. Embora o desempenho seja satisfatrio para uma implementao inicial, acredito que melhorias podem ser alcanadas ajustando hiperparmetros, aumentando o nmero de pocas de treinamento, ou utilizando tcnicas mais avanadas de Deep Learning, como arquiteturas de redes neurais mais profundas ou transfer learning.

Este projeto serve como uma base slida para aplicaes futuras em viso computacional, mostrando o potencial do Deep Learning para resolver problemas complexos de classificao de imagens.
